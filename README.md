# Missile-flight-model
6 DOF missile flight model usimg rigidbody3d. this missile system is highly customizable, and is easily integrated into your project. currently a work in progress.

details:
not only is this missile designed to be customizable, but i've worked to make it easy to modify as well and it's specifically designed to perform well.
on average, a missile with the stock settings will only cost you 0.09ms of frametime, assuming you have a ryzen 5 7600X desktop processor in a basic scene.
(performance settings are adjusted in the script.)

if you wish to learn more, clone this repo and open the example scene. press enter to launch a missile and click om your mouse to look around. the missile's export variable list is completely documented, so I won't write anything here. tinker a bit!
